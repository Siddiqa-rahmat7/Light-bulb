# Light Bulb

This is a simple light bulb project that demonstrates the use of HTML, CSS, and JavaScript to create an interactive web page.

## Key Features

*   **Toggle Light:** Click the button to turn the light bulb on and off.
*   **Visual Feedback:** The light bulb image changes to reflect its current state (on or off).
*   **Simple Interface:** A clean and straightforward user interface.

## File Descriptions

*   **index.html:** The main HTML file that provides the structure of the web page. It includes the light bulb image and the control button.
*   **style.css:** The CSS file that styles the components of the web page, such as the layout, colors, and appearance of the light bulb and button.
*   **bulb.js:** The JavaScript file that implements the logic for toggling the light bulb on and off. It handles the click event on the button and updates the image source accordingly.

## Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Siddiqa-rahmat7/Light-bulb.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Light-bulb
    ```
3.  **Open the `index.html` file in your web browser.**

## How to Use

*   Open the `index.html` file in a web browser.
*   Click the "Turn On" / "Turn Off" button to toggle the light bulb.
